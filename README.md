# ☕ Cafe Raw Tostador

Proyecto de análisis de datos centrado en el proceso de café en un entorno de tostador, trabajando desde datos en bruto (*RAW*) hasta su transformación y análisis.

---

## 📌 Descripción

Este proyecto simula el flujo real de datos dentro de un negocio de café especializado, donde se analizan procesos relacionados con la producción, tostado y venta del café.

Se trabaja con un enfoque de pipeline de datos:

RAW → CLEAN → ANALYSIS


El objetivo es practicar limpieza, transformación y análisis de datos aplicados a un caso realista del sector cafetero.

---

## 🎯 Objetivos del proyecto

- Trabajar con datos en bruto (raw data)
- Limpiar y estructurar datasets
- Aplicar procesos de transformación de datos
- Realizar análisis exploratorio (EDA)
- Extraer insights útiles del negocio
- Simular un flujo real de analista de datos

---

## 🧠 Contenido trabajado

- Limpieza de datos (nulos, duplicados, formatos)
- Transformación de variables
- Análisis exploratorio de datos (EDA)
- Agrupaciones y métricas
- Interpretación de resultados
- Preparación de datasets para análisis

---

## ☕ Contexto del proyecto

El proyecto está inspirado en el mundo del café de especialidad y el trabajo del **maestro tostador**, donde el control del proceso es clave para obtener un producto final de calidad.

Se simula cómo los datos pueden acompañar decisiones como:
- tipos de café más demandados
- evolución de producción
- rendimiento del proceso de tostado
- comportamiento de clientes
